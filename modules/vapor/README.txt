1. Getting started
==============================================================================

    A. Supported platforms

        VR Juggler is officially supported on the following platforms:

            IRIX 6.5
            RedHat Linux 6.1 and newer
            Windows NT 4.0 and 2000
            Windows 98/Me

        VR Juggler is known to work on the following platforms (but is
        not officially supported):

            HP-UX 10.20 and 11.00
            FreeBSD 4.2 and 5.0-CURRENT (x86)
            Sun Solaris 7 (x86)
            Mac OS X

    B. Minimum requirements (by operating system)

        * All
            GNU make 3.78 or newer
            GNU Autoconf 2.13
            GNU Automake 1.4

        * IRIX
            MIPSpro Compilers 7.3.1.1m or newer

        * RedHat Linux 6.x
            GCC 2.91

        * RedHat Linux 7.0
            GCC 2.96.69

        * HP-UX
            aCC A.03.30

        * Win32
            Microsoft Visual Studio 6.0 (SP4 or SP5)
            Juggler Tools (http://www.vrjuggler.org/pub/juggler-tools.zip)
