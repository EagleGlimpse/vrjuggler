Sim Position Chunk

Sim Position Chunk token: ``SimPosition''
Defines a simulated position device. It uses a user-specified keyboard proxy to accomplish this simulation.
Keyboard Proxy Property token: ``keyboardProxy''
Sets the proxy pointing to the keyboard to use for the simulation. The proxy must point to a KeyboardProxy chunk. More chunk types can be added through the chunk description file (chunksDesc).
Keys Property token: ``keys''
The list of keyboard keys, mouse movement directions and mouse buttons that can be used for changing the value returned by the simulated position device. All standard keys (except the F1-F12 keys), modifier keys (listed above), cursor movement keys, mouse movement directions and mouse buttons (for a three-button mouse) are available for use. Each key can be assigned to a particular aspect of the simulation, and modifiers can be used to allow the same keys (say those on the number pad) to be used for different aspects of simulation.
Modifier Keys Property token: ``modKeys''
Lists the modifier keys that may be used together with normal keyboard keys and mouse buttons when changing the value returned by the position device. The modifier keys must be chosen from a predefined set of keys that are treated as modifiers. The current modifier keys are:
  • NONE (no modifier)
  • SHIFT (the <SHIFT> key)
  • CTRL (the <CTRL> or <CONTROL> key)
  • ALT (the <ALT> or <META> key)
These keys and their values have special significance to VR Juggler and should not be altered.
Initial Position Property token: ``initialPos''
Sets the initial position of the simulated device using three floating-point values.
Initial Rotation Property token: ``initialRot''
Sets the initial Euler rotation of the simulated device using three floating-point values giving the x, y and z angles.
Delta Translation Property token: ``dtrans''
The floating-point translation change made with each key press.
Delta Rotation Property token: ``drot''
The floating-point rotational change (in degrees) made with each key press.
Translation
Coordinate System 		Property token: ``transCoordSystem''
A boolean value (0 for false, 1 for true) stating whether or not the translations apply to the tracker's local coordinate system.
Rotation
Coordinate System 		Property token: ``rotCoordSystem''
A boolean value (0 for false, 1 for true) stating whether or not the rotations apply to the tracker's local coordinate system.
