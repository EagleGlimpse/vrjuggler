NOTE: This applies only to the installation at ICEMT.  Other sites will have different directory structures.

These are the paths that need to be set for Juggler.

Environment Variables

setenv VJ_BASE_DIR /home/selab/Juggler/beta-1.0

setenv JDK_HOME /usr/java

Linking to Dynamic Shared Objects

For N32: setenv LD_LIBRARYN32_PATH $VJ_BASE_DIR/lib32
For 64: setenv LD_LIBRARY64_PATH $VJ_BASE_DIR/lib64
