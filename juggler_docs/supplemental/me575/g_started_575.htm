NOTE: This applies only to the installation on the Irix 6.5 eng machines.  Other sites will have different directory structures.

These are the paths that need to be set for Juggler.  I would recommend putting them in your shell configuration files.

Environment Variables

setenv VJ_BASE_DIR /home/ece/lockers/CprE575/Juggler/alpha_03

setenv JDK_HOME /usr/java

Search Path

set path = ( $VJ_BASE_DIR/bin $path )

Linking to Dynamic Shared Objects

For N32: setenv LD_LIBRARYN32_PATH $VJ_BASE_DIR/lib32
