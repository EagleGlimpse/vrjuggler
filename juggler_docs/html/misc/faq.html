General

What is VR Juggler?

VR Juggler is a C++ class library that is used as a framework for application development.  VR Juggler controls the low level aspects of the system that the user application is executing on.  It provides a generic interface to common input and output devices that VR Juggler can control

VR Juggler controls:

  • Graphics rendering
  • VR devices
  • Graphic projection setup
  • Multi-processing
  • Process synchronization
What VR Juggler is not

VR Juggler does not have it's own graphics API. It requires you to use a standard API such as OpenGL.

VR Juggler does not have any support for scene graphs or other data structures for the application graphics

VR Juggler is not a visual development environment.
