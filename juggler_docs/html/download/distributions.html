LiNUX

Requirements

* First, please Take Our Survey
RedHat 7.0, 6.2 (will probably work on other LiNUXs though not tested)
OpenGL support (Requires Mesa 3.1 or higher, 3D hardware recommended)
Performer 2.3 for LiNUX
Java 1.2 SDK or above (for VjControl)
NOTE: With RedHat 7.0, use gcc version 2.96.69 20000731 (use gcc -v to find your version), we've had problems with the default gcc that comes with RedHat 7.0

Libraries

Download Full LiNUX Binaries

  • Current Stable Version 1.0.0 (source only): vrjuggler-1.0.0.src.tar.gz (1.6 MB)

  • Version 0.1.97 (i386/RedHat6.2): vrjuggler-0.1.97.linux.i386.tar.gz (9.7 MB)

 

Win32

Requirements

* First, please Take Our Survey
WinNT 4.0
Visual Studio 6.0 SP4 (SP5 is recommended)
OpenGL support (hardware recommended)
Java 1.2 SDK (for VjControl)
juggler-tools-B20.zip package (for use with Cygwin B20) or juggler-tools-1.1.zip package (for use with Cygwin 1.1) for compiling and installing VR Juggler from source
Cygwin including their distributed Perl version (needed for compiling and installing from source)
