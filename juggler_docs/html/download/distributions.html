LiNUX

Requirements

* First, please Take Our Survey

  • RedHat 7.0, 6.2 (it will probably work on other LiNUX distributions, but it has not been tested)
  • OpenGL support (requires Mesa 3.1 or higher, 3D hardware recommended)
  • Performer 2.3 for LiNUX
  • Java 1.2 SDK or above (for VjControl)
NOTE: With RedHat 7.0, use GCC version 2.96.69 (20000731) or better (use gcc -v to find your version). We have had problems with the default GCC that comes with RedHat 7.0. All versions of GCC 2.96 are potentially unstable, especially with Performer-based applications.

Libraries

  • Version 0.1.97 (i386/RedHat6.2): vrjuggler-0.1.97.linux.i386.tar.gz (9.7 MB)

 

Win32

Requirements

* First, please Take Our Survey

  • Visual Studio 6.0 SP4 (SP5 is recommended)
  • OpenGL support (hardware recommended)
  • Java 1.2 SDK (for VjControl)
  • juggler-tools.zip (based on Cygwin 1.1.7) for compiling applications from the command line
  • Cygwin including their Perl version for compiling and installing the VR Juggler libraries from source.
    To get Perl, you may have to click the ``Exp'' option in the Cygwin Setup program.
