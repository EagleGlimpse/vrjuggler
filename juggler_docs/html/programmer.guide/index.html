Programmer's Guide

This book is for people interested in learning how to write a VR Juggler program.

The book contains the following topics:

  • Application objects
  • VR Juggler utility classes
  • Application tutorials
  • Advanced VR Juggler programming

Please note, the following prerequisites:

  • C++ experience (virtual functions, inheritance, etc)
  • OpenGL programming experience (for the OpenGL tutorials)
  • Performer experience (for the Performer tutorials)
  • Some knowledge of the VR Juggler architecture is needed for some advanced discussions.  See the Juggler Architecture guide
