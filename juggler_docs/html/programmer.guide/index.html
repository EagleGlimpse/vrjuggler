Programmer's Guide

Audience

This book is for people interested in learning how to write a VR Juggler based virtual reality application.

Required reading

Before you begin, note the following prerequisites:

  • C++ experience (virtual functions, inheritance, etc)
  • OpenGL programming experience (for the OpenGL tutorials)
  • Performer experience (for the Performer tutorials)
  • Some knowledge of the VR Juggler architecture is needed for some advanced discussions. 
    See the Juggler Architecture guide
