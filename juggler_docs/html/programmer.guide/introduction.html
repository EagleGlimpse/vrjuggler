Juggler Virtual Reality Platform - Introduction

Juggler is an exciting platform for virtual reality applications. You develop your application once for Juggler, then any where that Juggler is supported, your application is also supported.

VR juggler's design addresses several key requirements for a virtual reality development system.

  • Performance
  • Flexibility
  • Rapid application development
  • Ease of use
  • Extensibility
  • Portability

 

Currently VR Juggler is being used by several research institutions and is under active development
