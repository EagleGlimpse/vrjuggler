Environment Variables

Introduction

The environment variable VJ_BASE_DIR must be set to the base directory of the installed Juggler library.  This tells the kernel and the config editor where to find data files needed to run the system.

Ex. setenv VJ_BASE_DIR /home/software/Juggler/beta-0.1

In order to run the Java based config editor, the environment variable JDK_HOME must be set to tell the system where Java is located.

Ex. setenv JDK_HOME /user/java

List of All Juggler Environment Variables:

VJ_BASE_DIR

Set this variable to the directory where VR Juggler is installed.  This is used to find global data files.  Some makefiles also use this to find the library

VJ_DEBUG_NFY_LEVEL

This variable sets a filter on the amount of debug information for the library to print.  0 - Little or no debug output  7 - Full debug output

Warning: Setting this variable to a high value can cause a HUGE amount of text output.
