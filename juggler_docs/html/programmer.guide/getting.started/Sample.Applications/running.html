Running a Juggler application

All of the test programs take a config file as a parameter.  For example to run the cubes application a user would type:

cubes ~/.vjconfig/sim.base.config ~/.vjconfig/sim.displays.config ~/.vjconfig/sim.wand.mixin.config

Some of the applications need access to VJGlove, which is an alias for a glove device.  For these programs you also need to specify the ~/.vjconfig/sim.glove.mixin.config file.

If the program complains about not being able to find the runtime libraries it needs, see the section above about dynamic shared objects.
