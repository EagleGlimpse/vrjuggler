Installing Juggler

Most installations are either done by your system administrator, or (if none) then installed into a directory of your choice. We recommend a directory called "juggler"
If you downloaded the Source Code to juggler, you may have to compile it. Click here to learn how to do this.
  1. Go to the Juggler download page and get the latest version of the Juggler library in binary form.
  2. Unzip or Untar the file into a directory called "juggler".
  3. NOTE: Juggler is not set up yet!
** Move to the next topics in the tree control to learn how to compile and run Juggler applications.
