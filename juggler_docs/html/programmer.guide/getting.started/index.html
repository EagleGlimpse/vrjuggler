Getting Started

Just starting out with Juggler? Start here. This chapter will guide you to getting Juggler set up and running on your system.
Just use the tree control on the left side of your browser to change topics.


Juggler (and this guide) have a few pre-requisites:

Prerequisites:

  • Knowledge of C++ and object oriented design
  • Knowledge of one of Juggler's supported graphics API(s). (OpenGL Red Book, Performer manual, etc...)
