Configuration

Config files

Juggler requires config files in order to setup the system.  In the current version of the library, config files are passed to the application as a parameters.  By default, these config files are located in the users home directory in the ~/.vjconfig subdirectory. 

Each release starting with alpha 4 includes sample config files that may be copied to the users ~/.vjconfig directory to be used as a starting point for customization.   

Sample config files

The sample files are located in VJ_BASE_DIR/Data/configFiles.   (Where VJ_BASE_DIR is the base directory for the Juggler release being used.) 

In order to get started using Juggler quickly, you should copy all the config files from the configFiles directory into your local ~/.vjconfig directory.

NOTE: You are not required to use the default configuration files.  You are encouraged to customize them to suit your individual tastes and needs.

Editing config files

The juggler GUI controller (vjcontrol)  is used to edit the config files used by VR Juggler.
