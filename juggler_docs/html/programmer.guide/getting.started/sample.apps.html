Sample Applications

Use the sample applications supplied with Juggler to test and verify your installation. After that, you may want to edit them and get a feel for how they work. For a complete understanding of Juggler, be sure to read through the Juggler online books.

Juggler includes sample applications located in the "sample" directory in the installed Juggler area (VJ_BASE_DIR). Users can copy these files into their local home directories to modify and compile them.  Each application is in a separate directory and has a Makefile that will compile the application.

The directories that contain sample applications are:

  • ogl/wand
  • ogl/cubes
  • ogl/simpleGlove
  • ogl/torus
  • ogl/glove
  • ogl/combo
  • pfNav
