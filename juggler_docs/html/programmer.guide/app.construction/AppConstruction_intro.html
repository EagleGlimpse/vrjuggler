Application construction

This part of the documentation comprises the main bulk of information about application development.  The section is structured around the types of application that may need to be constructed.

The chapters each outline one area of interest for application developers.  For example, there are chapters that show how to get input from the system and chapters that show how to write applications for each of the currently supported graphics APIs.

Please note that when writing an application, there will be overlap between these chapters.  For example if you want to have input, sound, and OpenGL graphics in an application then your application will be based on concepts from each of those chapters.

  • VR Juggler application base: Getting and processing input from VR Juggler.
  • OpenGL applications: Writing an OpenGL based VR Juggler application.
  • Performer applications: Writing a Performer based VR Juggler application.
  • VTK based applications: Writing a VR Juggler application that uses VTK.
  • Juggler Sound Output: VR Juggler sound manager.
