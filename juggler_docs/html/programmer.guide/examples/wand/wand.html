Wand Position

Get the wand position
      vjMatrix* wandMatrix;
      wandMatrix = mWand->GetData();
These lines get a pointer to the matrix that holds the location of the wand.   mWand is a vjPosInterface object that was initialized to point to "VJWand".  When it is dereferenced ( mWand-> ) it evaluates to a pointer to a vjPosProxy.  This allows the user to call the GetData function of vjPosProxy which returns a pointer to a matrix with the location of the wand.
 

Draw a cube at the wand position
       glMultMatrixf(wandMatrix->getFloatPtr());
       glColor3f(1.0f, 0.0f, 1.0f);
       glScalef(0.25f, 0.25f, 0.25f);
       drawCube();
A vjMatrix object is in the same format that OpenGL uses for matrices.   Because of this, we can just call glMultMatrix( ourMatrix->getFloatPtr() ).   We need the getFloatPtr because OpenGL expects a pointer to 16 float values, not a matrix object. Note: In future versions there may be an automatic type conversion from matrix to float*.

Once the matrix has been passed to OpenGL, the box is drawn at the position of the wand.
