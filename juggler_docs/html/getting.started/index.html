Getting Started Guide

This book is for people who are just getting started with VR Juggler. It contains the following topics:

  • A high-level introduction to VR Juggler
  • Getting VR Juggler
  • Installing VR Juggler
  • Setting Up Your Environment
  • Compiling a Sample Application
  • Running a Sample Application
The prerequisites for reading this documentation are minimal. They are:
  • Some experience with a command-line interface (i.e., a shell such as tcsh or the DOS shell)
  • Creating and browsing directories
