Introduction

Of particular interest, The Juggler Documentation project uses a few Perl scripts to generate and install the documentation. In addition, Juggler uses Perceps to auto-generate it's C++ Class reference.
To better understand what you're doing, please understand the concepts listed in Required Reading.
Nieve posting about these subjects to the mailing list will probably be ignored.

Required Reading:
  • Perl documentation. (Try doing a 'man' or 'info' on perl, there also is a good book by O'Reilly.).
  • perceps documentation (Used in juggler to autogenerate documentation for CPP classes).
  • CVS documentation (used for revision control, lets you directly check in and out source)
