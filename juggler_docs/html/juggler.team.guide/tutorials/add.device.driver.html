How To Add a Device Driver To Juggler

First, decide what type of device you need to implement, there are a few to choose from:

  • vjDigital, vjSimDigital
  • vjAnalog, vjSimAnalog
  • vjPosition, vjSimPosition
  • vjGlove, vjSimDigitalGlove, vjGloveGesture

Lets take a look at a digital device. vjDigital derived devices give you a boolean 0 or 1 value. 

class MyDigitalDevice : public vjDigital
{
public:
   virtual void startSampling();
   virtual void stopSampling();
   
   virtual void sample();
   virtual void getData();
   
private:
   int mDigitalData;
};

void vjDigital::startSampling()
{
   // spawn a sample thread, which calls vjDigital::sample() repeatedly
}
   
// this is called repeatedly by your thread
void vjDigital::sample()
{
}

virtual void stopSampling()
{
   // kill sample thread
}

int vjDigital::getData()
{
   return mDigitalData;
}
