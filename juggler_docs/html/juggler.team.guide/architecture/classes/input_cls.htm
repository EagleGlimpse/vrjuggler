Overview

Input is managed by several components in VR Juggler.  These components work togethere to allow the application to work without being bound directly to specific devices.

Proxies

Proxies provide a placeholder to another object.  They present a general interface to a device in a way that abstracts that actual physical device interface.

Device Interface

A device interface is a "smart" object that talks to the input manager to get the application the correct proxy for the device name they specify.  The device interface is initialized with the name of a proxy or proxy alias to interact with.   This is the object that the user applications actually use to access the devices.
