Overview

Users need to derive their OpenGL applications from this class.  Then it is necessary to define any of the function from the vjApp class that are needed.  In addition the user needs to add their OpenGL drawing code the the member function draw().

Interface

(All functions of base class vjApp)
BaseApp Class

virtual void draw()

This function is called to draw the scene.  When it is called, the projection and modelview matrices will have already been set up with the correct viewing parameters.

The user should put all drawing code here.

virtual void contextInit()

This function is called immedately after a new context is created.  It is called for each new context.  Upon entry, the ogl context is set to the newly created context.  This function should be used to initialize any context specific information the user requires.  for example, display lists and texture objects are routinely initialized in this function.

virtual void contextPreDraw()

This function is called once per frame for each context.  When it is called, the OpenGL context is set to that of the current window being drawn.  This function can be used for any context specific changes the application needs to make.  This function should be used for things that need to happen every frame, but only once per context.   Ex: Dynamically Create display lists
