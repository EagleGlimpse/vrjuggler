vjDigitalProxy:  Digital data proxy interface

int GetData()

Returns the value of the digital input device.  Juggler digital devices support edge triggering.  This allows applications to not only know if a button is pressed, but also to detect when the button is pressed and when it is released. 

The value returned may have one of four values: 

Returned Value Meaning
vjDigital::OFF Button not pressed, and was not pressed last update either
vjDigital::ON Button on, and was on last frame as well
vjDigital::TOGGLE_ON Button was off, now it is on
vjDigital::TOGGLE_OFF Button was on, now it is going off

The identifiers are defined so that a simple test for non-zero means the button is pressed in some way. 

if(digProxy->GetData())      // Is button pressed
    doSomething();
else                                   // Button is not pressed
    doSomethingElse();

NOTE: Because of how TOGGLE_OFF is defined, testing for non-zero will result in a one update lag in detecting the button not being pressed
