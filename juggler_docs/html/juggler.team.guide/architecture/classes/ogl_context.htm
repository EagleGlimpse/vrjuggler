vjGlContextData

This class allows the user to specify a data type that contains data that needs to have a context specific copy. For example a struct full of display list id's. The user passes their user-defined data structure as the template parameter. The resulting object will be a "smart" pointer to the context specific data to use.

NOTE: Classes used as the user-defined data structure, must provide a default constructor used to initialize all of the copies of the data.

ContextDataType& operator*()
ContextDataType* operator->()

These are the pointer operations that are defined on this class.  They allow the object to be treated as a "smart" pointer within the application.

The references returned are only valid during the draw function and the context functions.  Results are un-defined for all other functions.
