vjKeyboardProxy: Keyboard device proxy interface

NOTE: The Juggler key identifiers are defined in vjKeyboard.h.   The format is VJKEY_X, where X is the symbolic name of the key.

int keyPressed(int keyId)

Returns the number of times the given key was pressed during the last frame of the application.

bool modifierOnly(int modKey)

Returns true if the given modifier key was pressed and no other modifier key was pressed during the last frame.

NOTE: The modifier keys are VJKEY_SHIFT, VJKEY_ALT, and VJKEY_CTRL
