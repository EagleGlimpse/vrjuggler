Device Proxies

Proxies provide a placeholder to another object.  They present a general interface to a device in a way that abstracts that actual physical device interface.

Proxies also allow Juggler to change devices at runtime.  Since a proxy provides an abstract interface to a device, the specific device being proxied can be changed at run-time.

There is on device proxy for each of the device types in Juggler:

  • vjPosProxy: Positional devices
  • vjDigitalProxy: Digital devices
  • vjAnalogProxy: Analog devices
  • vjGloveProxy: Glove based devices
  • vjGestureProxy: Gesture recognition devices
  • vjKeyboardProxy: Keyboard devices

NOTE: In addition to the information provided here, see the class documentation for each specific device proxy.
