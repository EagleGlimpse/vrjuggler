Overview

This class is the base class for all Performer applications.  As such it deals with Performer specific features.  Primarily this class adds the responsibility of handling a Performer scene graph.  It also gives users the ability to over ride the channel draw function.

Interface

virtual void initScene()

Initialize the Performer scene graph.  This is called after Performer has started.

virtual pfScene* getScene() = 0;

Return the current scene graph that the Performer application is using. This function must be defined so that the performer draw manager can get a scene graph to draw.

virtual void drawChan(pfChannel* chan, void* chandata)

Function called in the channel draw function to do the actual drawing

Advanced users may want to overide this behavior for advanced rendering effects such as over-lays, or multi-pass rendering.  See the Performer man pages about overiding the draw traversal function.  This function is the draw traversal function but with the projections set correctly for the given display and eye
