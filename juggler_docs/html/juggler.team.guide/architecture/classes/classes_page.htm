Primary Classes

VR Juggler is a very large system, but there are a few basic classes that the user needs to know about.

These can be divided into a few main sections

  • Kernel: The main control system of Juggler
  • Application: The classes used to encapsulate applications
  • Input: The classes used to get device input
  • Math: The basic math classes used in Juggler

Each page explains the general uses of the claass as well as gives a basic description of the major member functions and their uses.

For complete documentation on the classes and their interfaces, see the class reference.
