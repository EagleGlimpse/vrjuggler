VR Juggler Features

VR Juggler provides the following major features.

  • Virtual platform
    • Performance driven
  • Performance monitoring
  • Flexible and extensible
  • Modular
  • Run-time (re)configuration
  • Open Source
  • Cross-platform - Irix, Linux, Win32, others...
  • Cross-VR System - CAVE, HMD, Desks, simulators, others...
  • Configuration system
    • Parameterization of VR system components
    • GUI configuration editor - vjControl
