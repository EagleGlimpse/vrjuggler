JExplorer/JFinder JavaScript Commands

Version 1.08 - 11/1/2000
JExplorer has a few JavaScript assessable commands mainly for the selection and interrogation of the current selected item.
public void select( String text )
Select the first item with a given URL. This URL should be the full URL of a page including the protocol (ie. http://, ftp:// ... ). The URL searched is the left mouse button link attached to an item. Note the item selected is not activated only highlighted.
public String getSelect( )
Get the URL of the currently selected item. This is returned as a string and includes the protocol (ie http://, ftp://, .. ). If no item is selected this function will return null.
public void selectRef( int ref )
ref != 0
Select an item on the reference number attached to the item. The reference number is defined in the item's arguments and is provided as a way to keep track of items relative to database entries. All items are search and the first item with this reference number is set as selected. Note the item selected is not activated only highlighted.
public int getSelectRef( )
Get the reference number of the currently selected item. This is the value attached to the item when it is defined in the item tags and is returned as an integer value. If no item is selected this function will return 0.
public void setFile( int x )
x = 1 to n
Set file selects a script file from those specified in the applet tags file tags. (See applet file(n) ). This function tells the applet to load the script file specified or if loaded just display it. These files can give a different view to the same items or a simplified or expanded structure to the information. This different to a JIT script file as it doesn't load as a child of the current items but as a complete panel of items.
public void next( boolean expand )
public void prev( boolean expand )
expand = true or false
Next and Previous select the next or previous selected items from the tree. If no item is currently selected then nothing will happen. If expand equals true folders will be opened to select the first item otherwise the next visible
public void activate( void )
The public function activate will activate any link attached to the selected item.
