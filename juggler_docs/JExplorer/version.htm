JExplorer/JFinder Versions

Version 1.06 - 6/7/1999
Version 1.06 - 6/7/1999
  • Arrows tag for folders added to be compatible with JFinder.
Version 1.05a - 24/6/1999
  • Applet load adjusted to refresh on load to work on the Macintosh.
  • Hidelines added to remove lines between items.
  • Item adjusted to allow for a width and height of 0 so that the root item can be hidden.
Version 1.05 - 5/5/1999
  • Expanded Font tag arguments allow the modification of an items text colour, highlight color and mouse over color.
  • Default file argument set to html base page.
  • Links that are defined none or have no length are ignored.
  • Folders with no attached link will open on a single click otherwise they will require a double click as usual.
  • Background image slow loading problems resolved.
  • SelectURL and SelectID added to the parameters of treecontrol tag. These allows the setting of the inital item selected on loading of the applet based iether on the items URL or on its ref number.
  • JavaScript functions select, selectRef, getSelectRef, activate and getSelect added to the control to allow the selecting of items and the retrival of an items reference number or URL from JavaScript.
  • Showstatus flag added to the applet which removes all updates of the browser status bar. 
Version 1.02  - 24/2/1999
  • JavaScript function select added to select an item in the control from JavaScript. void select( URL );
  • autoclose parma tag added to applet.
  • Href in status string modified to remove machine name
Version 1.01 - 18/2/1999
  • plusColor parma tag added to modify the color of the plus and minus signs.
  • mouseunder parma tag color added added to modify color of the highlight underline for items.
  • title parma tag added to change initial root folder tag.
  • Fix for large folder scroll open and close which was to slow.
  • Fix for when the jit folder is being created, the tree control blanked out until the jit folder has been loaded.
